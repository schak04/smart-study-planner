# Smart Study Planner

This is a simple web app that helps students organise subjects, manage schedules, track progress, and improve productivity.

---

## Plans

### Tech Stack
- HTML, CSS, and vanilla JavaScript
- localStorage for data persistence

### Features to be implemented
Allow users to:
- add subjects
- plan study time
- set deadlines
- receive reminders
- analyse performance

### Application Sections
- **Dashboard:** Overview of subjects, deadlines, and daily schedule.
- **Subject Management:** Add, edit, delete subjects with priorities.
- **Schedule Planner:** Daily and weekly timetable with conflict handling.
- **Task Manager:** Assignments, exams, and deadline alerts.
- **Progress Analytics:** Reports, charts, and insights.
- **Settings:** Theme, preferences, data reset, and export.
